# Student Grade & Statistical Analysis System

## Description
This C program simulates a grading system for a university class. It demonstrates the use of **Parallel Arrays** to manage student data (IDs, Midterm, Final scores).

## Key Features
* **Weighted Calculation:** Calculates GPA based on 40% Midterm and 60% Final weight.
* **Pass/Fail Logic:** Automatically determines the status of each student (Passing grade: 50).
* **Statistical Analysis:** Implements an algorithm to calculate the Class Average and identify the Highest Performing Student (Best Score & ID).
* **Data Integrity:** Uses `float` types to ensure precision in decimal calculations.

## Technologies
* C Programming Language
* Standard I/O Libraries
